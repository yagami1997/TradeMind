# 数据目录 (Data Directory)

本目录用于存储和管理系统所需的各类数据。

## 目录结构

- `market_data/`: 市场数据
  - `stocks/`: 股票数据
    - `daily/`: 日线数据
    - `minute/`: 分钟数据
  - `indices/`: 指数数据
- `cache/`: 缓存数据
  - `technical/`: 技术分析缓存
  - `fundamental/`: 基本面数据缓存
- `downloads/`: 下载的原始数据

## 使用说明

1. 市场数据应按照类型和时间周期分类存储
2. 缓存数据会定期自动清理
3. 下载目录用于临时存储原始数据，处理后会移动到相应目录
